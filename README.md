## 🚀 How to develop this project

### Project setup

```bash
yarn install
```

### Compiles and hot-reloads for development

```bash
yarn dev
```

### Compiles and minifies for production

```bash
yarn build
```

### Lints and fixes files

```bash
yarn lint:fix && yarn format:fix
```
